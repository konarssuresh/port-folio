Neighbourhood Map Project

In this project I made a map of some places which i know in Chennai. The project implements MVVM model in which data is seperate from user interface and the user does not interact with data directly.For that purpose popular framework KnockoutJS is used.

The project also uses third party APIs such as google maps and mediaWiki(Wikipedia).Google map is used for showing the map and mark the places based upon lat-lng values of a particular place.Media wiki is used to display information about the place which is provided by wikipedia using AJAX.Filtering of places based upon text input is also done.and animation of marker is done when a particular place is selected.

Frameworks Used

  • KnockoutJS
  • Bootstrap

APIs used

  • google maps
  • Mediawiki
